🔧 実装方法

// テーマ切り替え
const theme = 'dark';
document.documentElement
  .setAttribute('data-theme', theme);

// システム設定を検出
const prefersDark = window
  .matchMedia('(prefers-color-scheme: dark)')
  .matches;

✨ 特徴

  • JavaScriptなしでも動作
  • CSS変数のみで実装
  • スムーズなトランジション
  • システム設定に対応
  • ローカルストレージで保存
