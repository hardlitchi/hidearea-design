# Grid（グリッド）

> レスポンシブなグリッドレイアウトを作成するためのグリッドレイアウトコンポーネント

**タグ**: `<ha-grid>`

**カテゴリ**: レイアウト

## プロパティ

| 名前 | 型 | デフォルト | 必須 | 説明 |
|------|------|---------|----------|-------------|
| `columns` | `number | string` | `12` |  | カラム数またはレスポンシブブレークポイントオブジェクト |
| `gap` | `'none' | 'xs' | 'sm' | 'md' | 'lg' | 'xl'` | `'md'` |  | グリッドアイテム間のギャップ |

## イベント

このコンポーネントはカスタムイベントを発行しません。

## スロット

| 名前 | 説明 |
|------|-------------|
| `default` | グリッドアイテム |

## 使用例

### 基本的な使い方

シンプルなグリッドレイアウト

```html
<ha-grid columns="3" gap="md">
  <div>アイテム 1</div>
  <div>アイテム 2</div>
  <div>アイテム 3</div>
</ha-grid>
```

---

### レスポンシブグリッド

レスポンシブなカラムを持つグリッド

```html
<ha-grid :columns="{ sm: 1, md: 2, lg: 4 }">
  <div>アイテム 1</div>
  <div>アイテム 2</div>
  <div>アイテム 3</div>
  <div>アイテム 4</div>
</ha-grid>
```

## アクセシビリティ

### ARIA ロール

特定のロールは定義されていません。

### キーボードサポート

標準的なキーボードナビゲーション。

### ARIA 属性

特定の ARIA 属性は不要です。

## デザイントークン

このコンポーネントは以下のデザイントークンを使用します:

### スペーシング

- `--spacing-xs`
- `--spacing-sm`
- `--spacing-md`
- `--spacing-lg`
- `--spacing-xl`
