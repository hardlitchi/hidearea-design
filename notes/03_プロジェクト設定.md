# hidearea-design プロジェクト設定

## プロジェクト概要

**プロジェクト名**: hidearea-design
**目的**: Web Componentベースのマルチフレームワーク対応デザインシステム

---

## パッケージ構成

### NPMパッケージ名

スコープ: `@hidearea-design`

```
@hidearea-design/core       # Web Components本体
@hidearea-design/react      # Reactラッパー
@hidearea-design/vue        # Vue 3ラッパー
@hidearea-design/tokens     # デザイントークン
```

### ディレクトリ構造

```
hidearea-design/
├── packages/
│   ├── core/           # @hidearea-design/core
│   ├── react/          # @hidearea-design/react
│   ├── vue/            # @hidearea-design/vue
│   ├── tokens/         # @hidearea-design/tokens
│   ├── docs/           # VitePress ドキュメント
│   └── storybook/      # Storybook
├── notes/              # プロジェクトメモ
├── .gitignore
├── package.json        # モノレポルート
├── pnpm-workspace.yaml # pnpm workspace設定
├── turbo.json          # Turborepo設定
└── tsconfig.json       # TypeScript基本設定
```

## 技術スタック

### 開発環境

- **Node.js**: v22.21.0
- **パッケージマネージャー**: pnpm 10.22.0
- **モノレポツール**: Turborepo
- **言語**: TypeScript
- **ビルドツール**: Vite

### デザイントークン

- **管理方法**: Style Dictionary
- **ソース形式**: YAML
- **出力形式**:
  - CSS Variables（メイン）
  - TypeScript型定義
  - JSON（ドキュメント用）

### スタイリング

- CSS Custom Properties（テーマ対応）
- Constructable Stylesheets（パフォーマンス）
- CSS Parts（外部カスタマイズ）

### テスト

- **ユニットテスト**: Vitest
- **E2Eテスト**: Playwright
- **ビジュアルリグレッション**: Chromatic

### ドキュメント

- **コンポーネントカタログ**: Storybook
- **ドキュメントサイト**: VitePress
- **API自動生成**: Custom Elements Manifest
