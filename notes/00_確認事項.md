## 推奨事項（現時点）

もし特に希望がなければ、以下のようなスタックを推奨します：

- **ビルドツール**: Vite + Turborepo（モノレポ管理）
- **言語**: TypeScript
- **デザイントークン**: Style Dictionary + JSON
- **テスト**: Vitest + Playwright
- **ドキュメント**: Storybook
- **スタイリング**: CSS Custom Properties + Constructable Stylesheets
- **最初のコンポーネント**: Button, Input, Checkbox（基礎的なもの）
- **ブラウザサポート**: モダンブラウザの最新2バージョン
